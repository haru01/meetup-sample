# Meetup Community Management

A full-stack TypeScript application for managing meetup communities and memberships.

## Tech Stack

- **Backend**: Express.js + Prisma + SQLite + TypeScript
- **Frontend**: (planned)
- **Testing**: Vitest + Supertest + Playwright

## Getting Started

```bash
npm install
cd backend
cp .env.example .env
npm run db:push
npm run dev
```

## Project Structure

```
backend/     Express API server
frontend/    (planned) React frontend
e2e/         (planned) Playwright E2E tests
```
